**Project Overview**



Build a secure serverless REST API using Amazon API Gateway, AWS Lambda, and DynamoDB. Amazon Cognito provides user authentication with JWT tokens. AWS WAF protects the API against common web exploits and rate-based attacks. CloudFront accelerates API delivery globally, while X-Ray enables distributed tracing across the application.



**Features**



* User Authentication
* JWT Authorization
* CRUD Operations
* DynamoDB Storage
* API Caching
* Distributed Tracing
* WAF Protection
* Global CDN
* Static Website Hosting
